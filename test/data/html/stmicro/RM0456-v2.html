10.10.38 PWR port I pull-down control register (PWR_PDCRI)

Each register bit PDy is protected against non-secure access when SECy = 1 in
GPIOI_SECCFGR.

Each register bit PDy is protected against unprivileged access when SECy = 1 in
GPIOI_SECCFGR and SPRIV = 1 in PWR_PRIVCFGR, or when SECy = 0 and
NSPRIV = 1.

Access: 14 AHB clock cycles added compared to a standard AHB access

Address offset: 0x94

Reset value: 0x0000 0000

(reset value not affected by exit Standby mode)
